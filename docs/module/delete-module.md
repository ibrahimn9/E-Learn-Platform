# Delete Module

**URL** : `{host}/api/v1/module/:moduleId`

**URL Parameters** : `moduleId=[integer]` where `moduleId` is the ID of the Cohort on the server.

**Method** : `DELETE`

**Auth required** : YES


**Permissions required** :

User Role Must be :

* Admin `AA`


## Success Response

**Code** : `204 No Content`

**Content example**

```json

```

## Error Response

**Condition** : If Database Throw Error.
**Code** : `400 Bad Request`
